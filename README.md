
# 할 일 

### 기능

- header, 로그인 라우팅 연결
- 상품 이미지 레퍼런스 모으기
- db.json 파일 만들기 
- json 서버로 데이터 요청
- localStorage에 회원정보 담기
- localStorage에 있는 값 화면에 출력
- 회원정보 수정시 localStorage에 있는 데이터 업데이트
- 로그아웃시 localStorage 데이터 삭제
- 상품 클릭시 해당 상품 디테일 정보 보이기
- 쿼리스트링에 검색한 벨류값 넣기
- 검색 결과값 출력
